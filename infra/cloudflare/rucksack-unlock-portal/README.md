# Rucksack Unlock Portal

Generated by Rucksack. This Cloudflare Worker gives GitHub and Discord pings a
browser page where a human can enter missing secret values without SSHing into a
VM. Secret values are used only in memory to write GitHub repository or
environment secrets, then discarded.

## Security Model

- Put this Worker behind Cloudflare Access before sharing the URL.
- Store the GitHub writer token as a Worker secret named `GITHUB_TOKEN`.
- Scope `GITHUB_TOKEN` to selected repositories and Actions secrets only.
- The Worker fails closed if `GITHUB_TOKEN` is missing and rejects cross-origin
  unlock form submissions before handling provided values.
- Do not put API keys, R2 keys, webhook URLs, or provider tokens in this repo,
  `wrangler.jsonc`, GitHub issues, Discord messages, logs, or manifests.
- `ALLOWED_REPOS` is required and must be a non-secret comma-separated allowlist
  of GitHub repositories this portal may unlock.
- `ALLOWED_EMAILS` is optional when Cloudflare Access policy already restricts
  users; set it as a non-secret comma-separated email allowlist for defense in
  depth.

Generated allowlist:

```text
ALLOWED_REPOS=erniesg/paillette
ALLOWED_EMAILS=<optional Cloudflare Access email allowlist>
```

## Setup

```bash
npm install
npx wrangler types
npx wrangler secret put GITHUB_TOKEN
npx wrangler deploy
```

If `ALLOWED_REPOS` still contains the placeholder or is empty, edit
`wrangler.jsonc` before deploying. The Worker fails closed until at least one
repository is allowlisted.

After deployment, configure Rucksack notifications to point at the portal:

```bash
export RUCKSACK_UNLOCK_BASE_URL=https://rucksack-unlock-portal.<account>.workers.dev
rucksack autopilot status erniesg/paillette --execute --notify-github
rucksack autopilot status erniesg/paillette --execute --notify
```

For one-off refreshes where the runtime env is not updated yet, pass the
non-secret Worker URL directly:

```bash
rucksack autopilot status erniesg/paillette --execute --notify-github --unlock-base-url https://rucksack-unlock-portal.<account>.workers.dev
```

Useful test URL:

```text
https://rucksack-unlock-portal.<account>.workers.dev/unlock?repo=erniesg/paillette&issue=20&secret=JINA_API_KEY&env_secret=CLOUDFLARE_API_TOKEN&app_env=ANVIL_R2_BUCKET&app_env=R2_ENDPOINT&environment=live
```
