# Agent State

This directory is the crash-recovery protocol for remote agents.

Tracked templates:

- `current-task.md`: current objective, branch, issue/PR, and next action.
- `latest-checkpoint.md`: what changed, what passed/failed, and how to resume.
- `decisions.md`: human decisions requested and answered.
- `failed-attempts.md`: approaches tried and why they failed.

Do not store secrets here.
