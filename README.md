# Paillette

Master repository for AI-powered educational platform and tools.

## Overview

This repository contains multiple integrated systems:

- **derivativ/** - Frontend React application (Vite + TypeScript)
- **derivativ.ai/** - Backend API service (FastAPI + Supabase) with multi-agent AI
- **gimme_ai/** - AI orchestration layer (Cloudflare Workers)
- **wanx/** - AI-powered video generation platform
